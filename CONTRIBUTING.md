# Руководство для участников (Contributing Guide)

Спасибо за интерес к проекту "Simulation_of_player_behavior"!

## Как внести вклад?
1.  Ознакомьтесь с **README.md**.
2.  Найдите задачу в разделе **Issues**, которую хотите выполнить, или создайте новую, обсудив идею.
3.  **Создайте ветку** для своей задачи.
4.  Внесите изменения, делая **понятные коммиты**.
5.  **Запушьте** ветку и создайте **Pull Request (PR)**.
6.  Обсудите изменения в PR, при необходимости внесите правки.
7.  После **одобрения** (approve) ваши изменения будут объединены (merged).

## Соглашения о коде
*   Следуйте стилю кода для Unity
*   Пишите комментарии на английском или русском языке.
*   Названия коммитов должны быть информативными (например, "Добавлен скрипт для первичной очистки данных", а не "правки").

## Создание Issues
Используйте шаблоны: `[Анализ]`, `[Данные]`, `[Документация]`, `[Исправления]`.

## Вопросы?
Задавайте вопросы в разделе **Discussions** этого репозитория.
